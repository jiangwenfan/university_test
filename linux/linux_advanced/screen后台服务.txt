1.install :
yum install epel-release
yum install screen -y 

2.
screen -S abc 创建一个abc的会话，打开了一个新的窗口。
在里面执行你想要执行的程序。再用Ctrl+a+d退出，让会话Detached，这样就能保证你的任务在后台一直运行，也不会随着终端的关闭任务就停止运行。

screen -ls #查看当前的会话。

screen -r abc #重新连接abc这个会话,让会话Attached

(4)进入会话，Ctrl+d 为退出并删除会话。或者screen -X -S 会话名称 quit 直接强制杀死一个Detached 会话。

screen -wipe #清除一系列Dead的会话。

ps -aux|grep screen #没有detached前可以看到多个screen进程。ctrl+a+d分离后，只能看到一个screen进程，但是每个screen会话里面的程序还是在正常执行。
screen -ls #可以看到目前正在后台运行的程序。attach表示在前台运行，关闭终端就会杀掉里面的进程。detached表示分离是后台运行的标志。
ps -aux|grep htop #比如你的screen会话里面运行的是htop程序，分离后，虽然只能看到一个screen进程，但是可以看到htop进程在正常运行。
